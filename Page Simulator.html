OS · MEMORY MANAGEMENT
FIFO — First In First Out

Evicts the oldest page in memory. Simple but can suffer Bélády's anomaly — more frames may cause more faults.

LRU — Least Recently Used

Evicts the page unused for the longest time. Good performance in practice; exploits temporal locality.

LFU — Least Frequently Used

Evicts the page with the lowest access count. Excellent for long-running workloads with stable hot pages.

OPT — Optimal (Bélády's)

Evicts the page not needed for the longest future time. Theoretical best-case baseline; requires future knowledge.
